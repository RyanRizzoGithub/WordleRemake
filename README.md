PURPOSE:
- This project is a recreation of the popular New York Time game "Wordle"

SPECIAL FEATURES:
- Original game music
- 3 unique modes (WOTD, Survival, DORDLE)
- 5 unique themes
- Global leaderboard
- Customizable player profiles
- Player statistics
- Ability to share your score
- Help page
- Animations


REQUIREMENTS:
- Have java installed
- Have SWT installed
